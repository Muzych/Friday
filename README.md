# Friday

Friday is a Telegram bot project built on top of [BUB](https://github.com/Muzych/bub).

## Project Status

This repository is freshly initialized as a Python project with `uv` and includes `bub` as a dependency.

## Quick Start

```bash
uv sync
uv run friday
```

## Notes

- The project uses Python 3.12.
- Dependency management is handled with `uv`.
- This repository is intended to grow into a BUB-powered Telegram bot.
